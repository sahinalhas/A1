### 3.2 API Endpoints:

#### Profil Skorları:
```
GET /api/student-profile/${studentId}/scores
```
Dönen Data:
- completeness: Profil tamamlanma yüzdesi
  * overall: Genel (%)
  * sections: Bölüm bazlı (%)
  * eksikAlanlar: Eksik alan listesi

#### Standartlaştırılmış Profil:
```
POST /api/standardized-profile/${studentId}/${endpoint}
```
Endpoints:
- academic
- health
- social-emotional
- talents-interests
- motivation
- goals-planning
- risk-protective
- behavior-incidents

#### Holistik Profil:
```
POST /api/holistic-profile/${endpoint}
```
Endpoints:
- strengths
- interests
- sel-competencies
- social-relations
- socioeconomic
- future-vision

#### Kariyer Rehberliği:
```
POST /api/career-guidance/analyze
POST /api/career-guidance/roadmap
GET /api/career-guidance/careers
GET /api/career-guidance/competencies/${studentId}
```
